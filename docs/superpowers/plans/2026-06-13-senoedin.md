# SenoedIn Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete hackathon-ready parody professional network "SenoedIn" ‚Äî LinkedIn-inspired, cat/climbing/gaming/music-themed, fully demo-mode with zero external API keys required.

**Architecture:** Next.js 16 App Router + TypeScript. All pages use localStorage via a storage abstraction. API routes have Gemini/YouTube opt-in behind env flags with deterministic fallbacks. Supabase scaffolded but not required.

**Tech Stack:** Next.js 16, React 19, TypeScript 5, Tailwind CSS v4, Lucide React, localStorage
